sound sensor

Implemented an electronic system that recorded and located the direction of a sound source using two microphones, servo motor that pointed in the sound direction, TIVA microcontroller, MyDAQ, Energia, and LabVIEW. Included a user interface that incorporated control and display features on LabVIEW, servo sound localization code using the TIVA microcontroller, stereo recording, timed recording, and audio band filtering code.
