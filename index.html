[eec10 final profect] :: Sound Sensor.

Implemented an electronic system that located and recorded the direction of a sound source using two microphones, servo motor that pointed in the sound direction, TIVA microcontroller, MyDAQ, Energia, and LabVIEW. Included a GUI that incorporated control and display features on LabVIEW, servo sound localization code using the TIVA microcontroller, stereo recording, timed recording, and audio band filtering code.

[discord bot] :: Frankki.

Frankki is a Discord bot created with JavaScript and deployed using Glitch.com. Specifically personalized to fit in with the servers between some friends and me, Frankki was designed to make our servers more interactive. Frankki's features include: modern-fantasy leveling up and health system, direct message greeting users joining a server, secret random messages, responses to certain words/phrases/inside jokes between some friends and me.

This bot is up and running, but still a work in progress! Future implementations are still in the making, so stay tuned! Started February 2019.
