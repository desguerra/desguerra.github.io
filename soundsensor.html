S O U N D S E N S O R .


sound sensor [EEC10 final project].

Implemented an electronic system that located and recorded the direction of a sound source using two microphones, servo motor that pointed in the sound direction, TIVA microcontroller, MyDAQ, Energia, and LabVIEW.

Learned to solder circuits. Included a GUI that incorporated control and display features on LabVIEW, servo sound localization code using the TIVA microcontroller, stereo recording, timed recording, and audio band filtering code. (2018)
